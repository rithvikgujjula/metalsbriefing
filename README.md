# MetalsDesk — Company Dashboards

Commercial intelligence dashboards for metals traders. Each company gets a widget grid: snapshot, full asset footprint (with capacity chart), market context, commercial angles, named counterparties, recent developments, tariff exposure, and financials — plus a live market tape powered by [OpenBB](https://openbb.co).

## How it works

- **Showcase dashboards** (`dashboards/*.json`): pre-baked, free to view, no API key. Ships with Nucor, Alcoa, Steel Dynamics, Cleveland-Cliffs, and Ryerson+Olympic Steel.
- **Generate any company**: visitor pastes their own Anthropic API key (browser-only), Claude researches with live web search and returns the dashboard as structured JSON.
- **The flywheel**: hit "Download JSON" on any generated dashboard, commit the file to `dashboards/` and add a line to `dashboards/index.json` — that company is now permanently free for everyone.
- **Market tape**: GitHub Action runs the OpenBB Platform daily (CME HRC futures, copper, aluminum, steel equities) → `market.json` → rendered as chips + HRC chart. Free, no keys.

## Repo layout

```
index.html                            the whole app
dashboards/index.json                 showcase list
dashboards/<slug>.json                one file per company
fetch_market.py                       OpenBB data fetcher
.github/workflows/update-market.yml   daily cron (place update-market.yml here)
market.json                           generated by the Action
```

## Deploy

1. Push `index.html`, `fetch_market.py`, the `dashboards/` folder, and `update-market.yml` (at `.github/workflows/update-market.yml`).
2. Settings → Pages → Deploy from branch `main`, root.
3. Actions tab → "Update market data (OpenBB)" → Run workflow once; it then runs weekdays 11:30 UTC.

## Cost

Showcase viewing: free. New company generation: ~$0.10-0.40 (Sonnet + web search), less on Haiku. Market tape: free.

## Notes

- API keys never leave the visitor's browser except in direct calls to api.anthropic.com.
- Paywalled assessments (SMU, Fastmarkets, RMDAS) are cited with dates in research bullets but not in the data tape.
- Showcase dashboards carry a `generated` date — regenerate periodically to keep them fresh.
- Research prompt and JSON schema live in `SYSTEM_PROMPT` inside `index.html`.
- AI-generated research: verify before trading or extending credit. Not investment advice.
